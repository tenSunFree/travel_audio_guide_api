## Tech Stack

- Go (Golang)  
  Statically typed, compiled language (High performance, simple deployment, and minimal runtime overhead for backend API services)
- net/http (Standard Library)  
  Built-in HTTP server & routing (Zero third-party dependencies, native request/response handling, and direct control over middleware composition)
- OpenAPI 2.0 (Swagger)  
  API contract & documentation (Standardized endpoint definitions, self-documented parameters, and structured response schemas)
- CORS & Logging Middleware  
  Cross-cutting concern handlers (Supports cross-origin requests and provides basic request tracing for debugging and observability)
